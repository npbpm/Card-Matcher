Comment utiliser?

Bonjour et bienvenue sur le CardMatcher du Groupe Image2 !
Ce programme va vous permettre d’enregistrer et de comparer des cartes de jeu à travers votre webcam. Pour cela, voici quelques instructions pour vous guider lors de votre utilisation.

    En cliquant sur le mode :
  • Vous pouvez maintenant créer votre propre base de données de cartes.
  • Placez une carte à jouer dans le rectangle vert et appuyez sur Apprentissage.
  • Afin de nommer la carte suivez ces normes : [numéro de la carte][couleur de la carte] Avec : Coeur = C / Carreau = D / Trèfle = T / Pique = P Et pour les têtes : Valet = V / Reine = Q / Roi = K / As = A
  • Les cartes que vous enregistrez se retrouvent alors dans le dossier “Apprentissage” qui constitue votre base de données personnelle. Les cartes enregistrées sur celle-ci le resteront même à la fin du programme.
  • Si une carte possède déjà le nom que vous choisissez, cette dernière sera écrasée.
  • Vous allez pouvoir tester la reconnaissance de cartes.
  • Commencez par choisir une base de données à l’aide du bouton “Select Database”. Cette dernière pourra être une base de données existante que vous avez téléchargé, ou par défaut la base de données Apprentissage que vous avez alimenté.
  • Placez une carte dans le rectangle rouge et appuyez sur Test.
  • Suite à cela, à droite de l’écran s’affiche alors la carte la plus proche de celle que vous venez de tester ainsi qu’un taux de correspondance.
  • Vous retrouverez l’image de la carte que vous testez au sein du dossier Test, et dans le dossier TestResult la comparaison de cette carte à toutes celles de la base de données choisie.
Bonne découverte!
